## 🎡 Geometric Theme Park
Technology: Minecraft Education Edition + MakeCode

## Technical Description
An algorithmic construction project using the Minecraft Agent to build parameterized geometric structures in a 3D environment. This project focuses on the automation of architectural forms through code.

## Key technical components:
Loops: Repetitive structures for symmetrical builds.
Numerical Parameters: Adjusting scales, heights, and radii dynamically.
Functions: Encapsulating specific build patterns (e.g., build_tower() or create_wheel()).
Spatial Control: Utilizing 3D coordinates (X, Y, Z) for precise placement.

## Core Concepts
Iterative Algorithms: Automating repetitive construction tasks.
Spatial Geometry: Applying 3D math to physical structures.
Parameterization: Designing code that adapts to different input values.
Logical Sequencing: Ensuring the correct order of blocks to maintain structural integrity.

## Developed Competencies
Spatial Visualization: Translating 2D logic into 3D objects.
Applied Mathematical Reasoning: Using geometry to solve construction challenges.
Practical Computational Thinking: Debugging spatial logic in real-time.
Code-to-Visual Correlation: Observing the immediate physical result of executed code.

## 📥 How to Import and Play (Step-by-Step)
Since this world was exported from Minecraft Education Edition (MEE), follow these instructions based on your version:

## 🎮 For Minecraft Education Edition (MEE)
Open Minecraft Education.
On the main screen, click Import.
Select the .mcworld file you downloaded.
Once imported, find the world in "My Worlds" and click Play.

## 💻 For Minecraft: Java Edition
Note: Java Edition does not natively support .mcworld files. You will need a conversion tool (like "Chunker.app") to convert it to a Java-compatible folder.
Convert the file to a Java folder using an online converter.
Press Win + R, type %appdata%, and go to .minecraft/saves.
Paste the converted folder into saves.
Open Minecraft Java, and the world will appear in your Singleplayer list.

## 📱 For Minecraft Bedrock (Console, Mobile, Windows 10/11)
Locate the .mcworld file in your file manager.
Double-click (or tap) the file.
Minecraft Bedrock will launch automatically and begin the import process.
Once finished, the world will be available at the top of your Worlds list.

## 📥 Como Importar e Jogar (Passo a Passo)
Como este mundo foi exportado do Minecraft Education Edition (MEE), siga estas instruções de acordo com a sua versão:

## 🎮 Para Minecraft Education Edition (MEE)
Abra o Minecraft Education.

Na tela principal, clique em Importar.

Selecione o arquivo .mcworld que você baixou.

Após a importação, encontre o mundo em "Meus Mundos" e clique em Jogar.

## 💻 Para Minecraft: Java Edition
Observação: o Java Edition não oferece suporte nativo a arquivos .mcworld. Você precisará de uma ferramenta de conversão (como o "Chunker.app") para convertê-lo em uma pasta compatível com Java.

Converta o arquivo para uma pasta Java usando um conversor online.

Pressione Win + R, digite %appdata% e acesse .minecraft/saves.

Cole a pasta convertida em saves.

Abra o Minecraft Java e o mundo aparecerá na sua lista de jogos para um jogador.

## 📱 Para Minecraft Bedrock (Console, Celular, Windows 10/11)
Localize o arquivo .mcworld no seu gerenciador de arquivos.

Clique duas vezes (ou toque) no arquivo.

O Minecraft Bedrock será iniciado automaticamente e começará o processo de importação.

Assim que terminar, o mundo estará disponível no topo da sua lista de Mundos.
